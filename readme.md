# 🖼️ Image Editor Web App

A simple and interactive **Image Editor** built using **HTML, CSS, and JavaScript**. This project allows users to upload an image, apply filters, use presets, reset changes, and download the edited image.

---

## 🚀 Features

- 📂 Upload Image from device  
- 🎨 Apply Filters dynamically  
- ⚡ Use Preset Filters  
- 🔄 Reset all changes  
- 💾 Download edited image  
- 🖥️ Clean and responsive UI  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla)  

---

## 📁 Project Structure
```
image-editor/
│── index.html
│── theme.css
│── style.css
│── script.js
```
---

## 📸 How to Use

1. Click **"Choose Image"** to upload an image  
2. Adjust filters using sliders  
3. Apply presets if needed  
4. Click **Reset** to remove changes  
5. Click **Download** to save the image  

---

1. Clone the repository:
